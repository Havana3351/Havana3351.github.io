与easy version的区别

两个版本题意完全相同，唯一的限制在输入数据的大小上，easy version 最大只有2000，而 hard version 有2*10^5。所以暴力肯定是不能用的！！

解题思路

我们可以考虑用DP，有三种开头就循环三次，每次都将字符串遍历一次，遇到不一样的就换，然后用数组记录从第一个字母到这个字母在这种开头情况下需要换多少次，最后输出时，用末尾字母要换的次数减去开头字母要换的次数，即可得到要求的子串长度答案。
