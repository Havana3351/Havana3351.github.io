搜索算法

搜索算法是利用计算机高性能来有目的的穷举一个问题的部分解空间或所有可能情况，从而求出问题的解的一种方法。今天主要来了解深度优先搜索（dfs）和广度优先搜索（bfs）。

深度优先搜索（dfs）

dfs属于图算法的一种，是一个针对图和树的遍历算法(Depth First Search)。利用深度优先搜索算法可以产生目标图的相应拓扑排序表，利用拓扑排序表可以方便的解决很多相关的图论问题，如最大路径问题等等。因为还没学过图论，所以我也不太清楚这句到底说了啥

基本步骤

一般用堆数据结构来辅助实现dfs算法。其过程可简要描述为对每一个可能的分支路径深入到不能再深入为止，而且每个节点只能访问一次。

DFS的实现方式相比于BFS应该说大同小异，只是把queue换成了stack而已，stack具有后进先出LIFO(Last Input First Output)的特性，DFS的操作步骤如下：

  1. 把根节点放入stack

  2. 重复下述3个步骤，知道stack为空为止：

    • 从stack中访问栈顶的点
    • 找出与此点邻接的尚且未遍历的点，进行标记，然后全部放入stack中
    • 如果此点没有尚未遍历的邻接点，则将此点从stack中弹出
