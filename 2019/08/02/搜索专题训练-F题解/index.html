HDU-Prime Ring Problem

vjude

题目详情

A ring is compose of n circles as shown in diagram. Put natural number 1, 2, …, n into each circle separately, and the sum of numbers in two adjacent circles should be a prime.

Note: the number of first circle should always be 1.

展开查看

Input

n (0 < n < 20).

Output

The output format is shown as sample below. Each row represents a series of circle numbers in the ring beginning from 1 clockwisely and anticlockwisely. The order of numbers must satisfy the above requirements. Print solutions in lexicographical order.

You are to write a program that completes above process.

Print a blank line after each case.

解题思路

一看到这个题就想到要用dfs，主要就是维护一个数组用来存放结果，维护另一个数组用来查询这个数是否被用过。这个dfs除了基本的需要注意的事项以外还要注意递归次数问题。

因为题目要求需要打印所有所有解，只有第一个位置可以固定为1，所以我们在最初在main函数中调用dfs时，就要将几种数字全搜索一遍。然后再dfs的递归调用中需要注意，找到一个答案后，不要急于返回开头，还要再换数字进行搜索，因为2个不同答案之间有可能某部分是相同的。不要问我为什么知道这些，当然都是我第一次想出来的其实是血的教训得来的
